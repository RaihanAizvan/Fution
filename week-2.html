Tips

  1. Give importance to Javascript over Bootstrap in this week.
  2. But Still dont dive deep into javascript because we will dive deep into it next week.
  3. Don't forgot to add comments to your code
  4. Use styles like Carousel, hover effects, Tooltips etc. in your website
  5. Portfolio must include 6 pages. That doesnt mean 6 html files. 6 different sections and link to those sections in the nav is enough
  6. Do not skip Javascript Because it will also useful in week 3
  7. Form Validation must check for all input types, for eg if user enters "$123username" as his name, we doesnt validate it properly. Reviwer might ask to input like this. So Please ensure that this is completly validated.
  8. Same for email. A proper email id is requried. Other wise throw a popup box or a messsage that indicates the form is not completely validated.
  9. Use EmailJS to send email. It is more easy than others.
  10. Performance is a required thing, We all skip that because we dont know how to handle it. If reviwer may asks for check the live Performance. If it is less than 90, they may give us 2 minute to increase the perfomance. If we cant, score get reduced.
  11. Perfomance of a website depent on the some contents. If you got stuck in low perfomance, dont worry about it.
    • Optimize images by compressing and resizing. you can use TinyPng or ImageOptim to reduce file size.
    • Minify CSS and HTML files. Use cssnano to minify css and use Html Minifier to minify html as well
    • Identify and remove unused CSS rules. Tools like PurgeCSS can be used to remove unused CSS, particularly if you are using Bootstrap.
    • Enable browser caching
    • Implement lazy loading for images and videos. Add loading = "lazy" attribute to you img tags.

      <img src= "image.jpg" alt= "Description"  loading = "lazy" >
  13. Feel free To answer any question, be prepared. Some reviwer may ask bootstrap only while others focus on javascript. It depends upon the reviwer.
