WEEK--01

HTML & CSS

Topics

  • Understand what HTML stands for (Hypertext Markup Language)

    • HElloo

  • Learn about the basic structure of an HTML document (DOCTYPE, <html>, <head>, and <body>)

  • Explore self-closing tags (e.g., <img>, <input>)

  • Familiarize yourself with different input types (text, password, checkbox, radio, etc.)

  • Understand the concept of content, padding, border, and margin in the CSS Box Model

  • Learn how to calculate the total width and height of an element

  • Study various selectors (class, ID, element, attribute, pseudo-class, pseudo-element)

  • Differentiate between position: absolute, position: relative, position: fixed, and position: sticky

  • Explore the z-index property

  • Know about different units (px, em, rem, %, vh, vw)

  • Understand how specificity affects style application in CSS

  • Learn about the universal selector " * "

  • Understand when and how to use the CSS !important Rule (sparingly)

  • Practice using margin shorthand (e.g., margin: 10px;, margin: 10px 5px 15px;, etc.)

  • Explore the opacity property

  • Combine selectors for efficient styling

  • Learn about display: flex and its properties (justify-content, align-items, etc.)

  • Understand the importance of the alt attribute in <img> tags

  • Create links using <a> tags

  • Understand the target="_blank" attribute

  • Learn how to add a favicon to your website

  • Explore common meta tags (e.g., <meta charset="UTF-8">, <meta name="viewport" content="width=device-width, initial-scale=1.0">)

  • Use semantic tags like <header>, <nav>, <main>, <article>, etc.

  • Understand the differences between inline, block, and inline-block display types

  • Set the character encoding in your HTML to UTF-8

  • Learn about combinators like descendant ( ), child (>), and adjacent sibling (+)

  • Explore CSS transformations (e.g., rotate, scale, translate, etc.)

  • Create tables using <table>, <tr>, <td>, etc.

  • Understand form elements (<form>, <input>, <textarea>, <select>, etc.)

  • Embed multimedia content using iframes, videos, and audio

  • Optimize for different screen sizes with the viewport meta tag

  • Use comments to explain your code

  • Achieve aligning two items in a row using flexbox or other techniques

  • Understand the importance of accessibility and semantic HTML

  • Learn about responsive web design principles

  • Explore CSS grid layout for advanced page structuring

  • Stay updated with the latest web development trends and technologies
